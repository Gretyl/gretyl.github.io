# gretyl.github.io

Personal GitHub Pages site with interactive tools and experiments.

## What's here

- [Artemis Trail](artemis-trail.html) — interactive lunar mission simulator
- [WebGL Shadertoy harness](tools/shadertoy.html) — shader IDE with `@slider` annotations for live parameter tuning

## Development

Static HTML and vanilla JS — no build step, no dependencies. Push to `main` and GitHub Pages deploys automatically.
