Harnessing Chris Okasaki's *Purely Functional Data Structures*, this Python module brings immutable, persistent versions of six foundational structures, from the classic ConsList and BankersQueue to advanced heap and tree variants such as LeftistHeap, PairingHeap, and RedBlackTree. The design emphasizes protocol conformance and Pythonic idioms, supporting safe, efficient operations where each mutation returns a new structure. Unlike mature libraries like [pyrsistent](https://github.com/tobgu/pyrsistent) or [immutables](https://github.com/MagicStack/immutables), this implementation uniquely provides functional heaps, amortized queues, and balanced trees absent elsewhere in Python's ecosystem. Rigorous testing covers structural invariants and API fidelity, with lazy evaluation (via Suspension/memoization) underpinning efficient queue operations. The module thus fills a key gap for magicians-programmers seeking true persistence over a broad range of data structures.

**Key findings:**
- Okasaki's heap, queue, and tree algorithms adapted natively to Python with idiomatic interfaces.
- Persistent and immutable semantics enable safe structural sharing; all mutations return new objects.
- Existing Python collections lack these advanced functional variants, especially heaps and balanced BSTs.
