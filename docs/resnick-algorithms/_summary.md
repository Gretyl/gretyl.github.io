Resnick Algorithm Artifacts is a collection of interactive, testable simulations that showcase how simple agent rules produce complex emergent behaviors, inspired by Mitchel Resnick’s *Turtles, Termites, and Traffic Jams*. Each artifact—such as flocking, termite wood gathering, and forest fire percolation—is built on a strict separation between pure JavaScript algorithm modules (deterministic, no DOM, seeded RNG) and browser-based UI (canvas rendering, controls). Automated tests use Node.js's built-in [`node:test`](https://nodejs.org/api/test.html) framework with zero dependencies, ensuring conservation laws, boundary conditions, and determinism across seven microworlds. Artifacts are designed for direct browser use, with live parameter tuning and statistical readouts, and follow an optional-toolchain philosophy where all test infrastructure is nonintrusive. For additional context, see the [magicians-programmers comparison](magicians-programmers-comparison/README.md) for the project’s creative framing.

Key findings:
- Emergent behaviors—including flocking, spontaneous traffic jams, pile convergence, and population oscillations—are consistently reproducible via simple agent rules and deterministic tests.
- Pure ES module design enables seamless reuse between browser artifacts and headless testing.
- Conservation properties and threshold phenomena are rigorously verified in all algorithms.
